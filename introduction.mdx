## Quick Start

Get up and running with Dif in minutes. Follow these steps to create your first banking API integration.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Create an account at https://dashboard.example.com/signup. Verify your email to access the developer console.
  </Step>
  <Step title="Generate API Key" icon="key">
    Navigate to the API section and generate your key.

    ```
    YOUR_API_KEY
    ```
  </Step>
  <Step title="Make Your First Request" icon="play">
    Test the accounts endpoint:

    <CodeGroup tabs="JavaScript,cURL">
      ```javascript
      const response = await fetch('https://api.example.com/v1/accounts', {
        headers: {
          'Authorization': `Bearer ${YOUR_API_KEY}`
        }
      });
      const data = await response.json();
      console.log(data);
      ```
      ```bash
      curl -H "Authorization: Bearer YOUR_API_KEY" \
        https://api.example.com/v1/accounts
      ```
    </CodeGroup>
  </Step>
</Steps>
